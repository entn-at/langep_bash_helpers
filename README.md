# BASH Helpers

A collection if reusable BASH functions and script template(s).

`utils.sh`: Helper functions
	- Can be used by sourcing it from a bash script.
	```source utils.sh```
	- Example: `new_script_template.sh`

`new_script_template.sh`: Bash script template. Can be used to start new bash scripts.
`run_test.sh`: Runs some unit tests against the helper function file.
